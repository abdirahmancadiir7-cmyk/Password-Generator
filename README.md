# Password Generator

A simple password generator built with Python. This project creates strong and random passwords using uppercase letters, lowercase letters, numbers, and special characters.

## Features

- Generate secure random passwords
- Custom password length
- Uses Python built-in libraries
- Fast and lightweight

## Requirements

- Python 3.x

## Installation

Clone the repository:

```bash
git clone https://github.com/abdirahmancadiir7-cmyk/password-generator.git
```

Move into the project directory:

```bash
cd password-generator
```

Run the program:

```bash
python3 generator.py
```

## Example

```text
Enter password length:
16

Generated Password:
R@7xP!9#Lm2$Qa8Z
```
